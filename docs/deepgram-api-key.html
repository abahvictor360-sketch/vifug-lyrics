AI auto-follow · setup guide

Get a Deepgram API key

Vifug’s AI auto-follow advances your slides by listening to the room. The speech-to-text runs on Deepgram, so you’ll paste one free API key into Vifug to switch it on. It takes about three minutes.

Why a key? Auto-follow streams a few seconds of microphone audio to Deepgram to turn speech into text, then matches it against your upcoming slides. Deepgram gives new accounts free credit — plenty for regular services — and your key is stored only on your own machine.

Create your key

  1. Sign up for a free Deepgram account

    Go to console.deepgram.com/signup and register with your email or a Google/GitHub account. New accounts include free usage credit — no card required to start.

  2. Verify your email and open the Console

    Confirm the verification email, then sign in to the Deepgram Console — the dashboard where you manage keys and usage.

  3. Create a new API key

    In the Console, open:

    API Keys→Create a Key
    • Give it a name you’ll recognise, e.g. Vifug auto-follow.
    • Leave the default Member permissions — that’s all Vifug needs.
    • You can leave the expiry and usage limits unset.

  4. Copy the key immediately

    Deepgram shows the full key only once. Copy it now and keep it somewhere safe. If you lose it, just delete that key and create a new one.

Add it to Vifug Lyrics

  1. Open the AI auto-follow settings

    In the Vifug operator, open Settings and go to:

    Settings→General→AI auto-follow

  2. Paste your key and pick the language

    Paste the key into the Deepgram API key field. Set Spoken language to the language your congregation sings in (or “Multi” to auto-detect), and tune the sensitivity if you like.

  3. Turn auto-follow on

    Toggle AI auto-follow on. When you send a slide live, Vifug starts listening and advances on its own — and your manual next / previous always overrides it. The status line shows “Listening…” with what it heard.

Common questions

Is it free?
Deepgram gives new accounts free credit that covers typical church use, and auto-follow only sends audio while a slide is live. Check current pricing on deepgram.com/pricing.
Where is my key stored?
In Vifug’s local settings on your own computer. It’s sent directly to Deepgram to open the live transcription connection and is never shared anywhere else.
Does auto-follow need internet?
Yes — only this feature does, because the speech recognition runs on Deepgram’s service. Everything else in Vifug (songs, all Bible versions, projector, NDI) stays fully offline.
Which languages work?
Deepgram supports many languages; pick the closest match in Settings, or choose “Multi (auto-detect)” to follow code-switching between languages. Recognition quality varies by language.
I lost my key — what now?
In the Deepgram Console, delete the old key under API Keys and create a new one, then paste the new key into Vifug. Old keys can’t be shown again for security.
