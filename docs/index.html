Streaming & NDI

One output, every screen

Every surface renders the same live state, kept in sync by the built-in server — nothing drifts out of step.

  • Transparent /stream page composites cleanly over your video feed in OBS or vMix.
  • Auto-reconnecting sync (Server-Sent Events) survives network hiccups.
  • Real NDI output via OBS + the free DistroAV plugin — vMix, TriCaster and Resolume see it instantly.
  • Backgrounds (image, video, color) flow through to the stream automatically.

Questions

Frequently asked

Is Vifug Lyrics really free?

Yes. It's free to download and use, with no watermark and no subscription. The source is on GitHub.

Does it work without internet?

Completely. Songs, all seven Bible versions and your settings live in a local database on your machine. Church Wi-Fi going down will never take your lyrics down.

Which platforms are supported?

Windows (installer), macOS (DMG) and Linux (AppImage). Grab the latest build from the download buttons above or the releases page.

How do I get lyrics into OBS or vMix?

Copy the stream URL from the operator's "Stream / OBS source" panel and add it as a Browser source (1920×1080, transparent). For NDI, enable OBS's NDI output via the DistroAV plugin — full guide in docs/STREAMING_AND_NDI.md.

Can I import my existing ProPresenter library?

Yes — .pro6 files import directly (plus best-effort .pro), and plain .txt / .docx lyrics are auto-detected into verses and choruses.

Can someone control slides from their phone?

Yes. Open the remote page on any phone or tablet on the same network and cue, advance or blank slides — the operator's machine stays in charge.
