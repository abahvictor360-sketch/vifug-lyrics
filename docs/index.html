Questions

Good to know

Is Vifug Lyrics really free?
Yes — free to download and use, no watermark and no subscription. The source is on GitHub.
Does it work without internet?
Completely. Songs, all seven Bible versions and your settings live in a local database on your machine. Church Wi-Fi going down will never take your lyrics down.
Which platforms are supported?
Windows (installer), macOS (DMG) and Linux (AppImage). Grab the latest from the buttons above or the releases page.
How do I set up AI auto-follow?
Auto-follow uses Deepgram for live speech recognition. Follow the Deepgram API key guide to get a free key, then paste it into Settings → General → AI auto-follow.
How do I get lyrics into OBS or NDI?
Enable native NDI in Settings → General, or add the stream URL as a Browser source in OBS (1920×1080, transparent). Full guide in STREAMING_AND_NDI.md.
Can I import my ProPresenter library?
Yes — .pro6 files import directly (plus best-effort .pro), and plain .txt / .docx lyrics are auto-detected into verses and choruses.
